# Instrucciones para Codex

- No implementar todo de golpe.
- Trabajar por etapas.
- Mantener backend y frontend separados.
- Backend con FastAPI, SQLAlchemy y PostgreSQL.
- Frontend con React, Vite y Bootstrap.
- No tocar módulos ya funcionales sin avisar.
- Explicar cada cambio importante.
- Priorizar primero estructura, luego login/roles, luego módulos, luego OCR, luego diseño visual.
